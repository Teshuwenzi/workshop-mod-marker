# workshop-mod-marker
Steam创意工坊小标记Visual marker &amp; note tool for Steam Workshop mods
浏览 Steam 创意工坊时，用来给 MOD 做标记和备注的小工具。
英语看不太懂？它可以给每个 MOD 写一句自己的备注；
这个MOD看过了？那个MOD不喜欢？点一下就能清楚地标出来！

A small userscript to mark and annotate Steam Workshop mods while browsing.

- Mark mods as viewed or disliked
- Add short personal notes for each mod
- Import / export all records as a TXT snapshot
- All data stays in your browser (localStorage)

Designed for people who browse a lot of mods and forget everything they saw.

## Install
- Install Tampermonkey
- Enable developer mode
- Install the script from Greasy Fork
